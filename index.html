Conectar la base de datos (una sola vez)

Este juego necesita una base de datos gratuita de Firebase para que los 100 celulares vean la misma partida en tiempo real. Crea un proyecto gratis en firebase.google.com, activa "Realtime Database" y pega aquí los 5 datos que te da la consola (Configuración del proyecto → tus apps → SDK de Firebase). Solo lo tienes que hacer una vez: se guarda en este navegador y queda incrustado cuando subas el archivo a tu hosting.
